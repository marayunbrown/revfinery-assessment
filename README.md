# Revfinery Assessment Tool

A free assessment tool that helps:
- **Companies** diagnose their revenue system health
- **Individuals** evaluate their sales skills

Built with Next.js and React. Deployed on Vercel.

## Live Site

[assessment.revfinery.com](https://assessment.revfinery.com)

## Features

- Two assessment tracks (Company & Individual)
- Score-based personalized recommendations
- Email capture for lead generation
- Mobile responsive design
- Branded to Revfinery design system

## Local Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Deployment

Deployed automatically via Vercel on push to `main` branch.
